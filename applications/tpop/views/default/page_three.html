What is a framework?

a framework is a real or conceptual structure intended to serve as a support or guide for the building of something that expands the structure into something useful.

What is a web-framework?

A web application framework (WAF) is a software framework that is designed to support the development of dynamic websites, web applications, web services and web resources. The framework aims to alleviate the overhead associated with common activities performed in web development.

Web2py vs Django

web2py is more compact and has more functionality out of the box (more supported databases, web based IDE, ticketing system, runs on GAE without patches, multiple login methods, role based access control, web based translation interface, support for multiple database connections, distributed transactions, and more).

Web2py is definitely easier and makes the code simpler and cleaner. But Django gives you a better understanding on whats going on, and also more power to modify the design/process.

Web2py vs Flask

Flask does not have a WYSIWYG editor unlike web2py.

Flask does not have static typing support and also does not have SQL injecting support unlike web2py

Explain MVC with an example

* Data (Model) * An interface to view and modify the data (View) * Operations that can be performed on the data (Controller)

Consider an address book.The model is a list of objects of the type "PERSONS".
The view is a GUI windows that shows the full address book.
The controller allows us to add/modify the contacts.
